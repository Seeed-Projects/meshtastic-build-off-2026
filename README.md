# Meshtastic Build-Off 2026

Official showcase repository for Meshtastic Build-Off 2026 projects.

This repository is intended to collect, review, and showcase community builds that use Meshtastic or Meshtastic-compatible hardware, firmware, enclosures, integrations, tools, or field deployments.

## Repository Structure

- `submissions/` - participant project submissions
- `showcase/` - curated project pages for public browsing
- `docs/` - event rules, judging notes, and organizer documentation
- `assets/` - shared images and media used by showcase pages
- `.github/ISSUE_TEMPLATE/` - GitHub issue forms for project submissions

## Submit a Project

Open a new project submission issue and include:

- project name
- short summary
- hardware used
- firmware or software links
- build photos or demo video
- team or author names
- license and reuse notes

After review, accepted projects can be added under `submissions/` and promoted into `showcase/`.

## Suggested Submission Layout

Create a folder under `submissions/` using a lowercase, hyphenated project name:

```text
submissions/example-project/
  README.md
  images/
  firmware/
  hardware/
```
